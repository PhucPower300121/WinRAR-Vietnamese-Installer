# WinRAR Vietnamese Edition

An open-source, customized installer tool designed to simplify the installation of WinRAR with full Vietnamese language support.

## Overview
This project provides a streamlined installer for WinRAR (v7.20), featuring:
- **Automatic Vietnamese Language Integration:** No more manual `.lng` file copying.
- **Optimized Configuration:** Ensures the software is correctly configured upon installation.
- **Smart Process Handling:** Automatically detects and safely terminates running WinRAR instances to prevent installation conflicts.
- **User-Friendly Setup Wizard:** A clean, transparent installation process that respects RARLAB's licensing while adding value for the Vietnamese user community.

## Why this project?
Installing WinRAR and manually patching it with language files can be tedious. This installer automates the entire process, making it accessible to everyone. 

> **Note:** This is a non-official, third-party installer project. I do not claim ownership of the WinRAR software itself. The WinRAR binary remains the property of win.rar GmbH.



## Features
- **Clean Installation:** No bundled bloatware or adware.
- **Safe Patching:** Uses precise process termination to ensure system stability.
- **Transparency:** Open-source script (`.iss`) allows you to audit exactly how the software is installed.
- **Easy Setup:** Intuitive wizard interface.

## How to use
1. Download the latest release from the [Releases page](#).
2. Run the installer as Administrator.
3. Follow the on-screen instructions.
4. WinRAR will be installed and automatically configured in Vietnamese.

## Security & Transparency
- **Source Code:** You can view the full Inno Setup script in the repository.
- **Checksums:** Verify the integrity of the installer by checking the SHA-256 hash provided in each release.
- **No Modification:** The original WinRAR binary (`.exe`) is not modified; only configuration files and language packs are managed by this installer.
